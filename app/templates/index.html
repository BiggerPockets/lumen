How to connect

  1. 1

    Create an org

    Visit /setup, enter your org name, and copy the API key shown.

  2. 2

    Add the hook to your repo

    Add this to your project's .claude/settings.json and commit it — all devs pick it up automatically:

    {
  "hooks": {
    "PostToolUse": [{
      "matcher": "Skill",
      "hooks": [{
        "type": "command",
        "command": "skill=$(cat | python3 -c \"import sys,json; d=json.load(sys.stdin); print(d.get('tool_input',{}).get('skill','unknown'))\" 2>/dev/null || echo unknown); curl --max-time 2 -s -X POST {{ base_url }}/events -H 'Content-Type: application/json' -H 'Authorization: Bearer $LUMEN_API_KEY' -d \"{\\\"event\\\": \\\"skill_used\\\", \\\"properties\\\": {\\\"skill\\\": \\\"$skill\\\"}, \\\"project\\\": \\\"$(basename $(git rev-parse --show-toplevel 2>/dev/null) 2>/dev/null)\\\"}\" > /dev/null 2>&1 &"
      }]
    }]
  }
}
  3. 3

    Set the API key in each dev's shell

    Each developer adds export LUMEN_API_KEY=your_key to their ~/.zshrc (or equivalent).

  4. 4

    Open your dashboard

    Visit /login, paste your API key, and events will start appearing as your team uses Claude Code.
